
## How to Create Video Player

A simple, accessible and customisable media player for  Video,  Audio,  YouTube and  Vimeo. for  [Main Github Video Sourch](https://github.com/sampotts/plyr)

You Need to skep anything using dot (.) Symbol

```bash
  <script src="https://cdn.plyr.io/3.7.8/plyr.js"></script>
```
OR 
```bash
  <script src="https://cdn.plyr.io/3.7.8/plyr.polyfilled.js"></script>
```
CSS
Include the plyr.css stylesheet into your <head>.

```bash
  <link rel="stylesheet" href="path/to/plyr.css" />
```
OR

```bash
  <link rel="stylesheet" href="https://cdn.plyr.io/3.7.8/plyr.css" />
```
